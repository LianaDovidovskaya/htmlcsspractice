Примеры использования тегов

Теги dl,dt,dd

Элементы dl, dt, dd предназначенны для создания списка определений.
Каждый такой список начинается с контейнера dl, куда входит тег dt создающий термин и тег dd ,
задающий определение этого термина. Закрывающий тег /dd не обязателен, поскольку следующий тег сообщает о завершении предыдущего элемента.
Тем не менее, хорошим стилем является закрывать все теги.
Пример использования:

<dl>
<dt>Термин 1</dt>
<dd>Определение термина 1</dd>
<dt>Термин 2</dt>
<dd>Определение термина 2</dd>
</dl>

Термин 1
Определение термина 1
Термин 2
Определение термина 2

Тег pre

Элемент <pre> определяет блок предварительно форматированного текста. Такой текст отображается обычно моноширинным шрифтом и со всеми пробелами между словами.
По умолчанию, любое количество пробелов идущих в коде подряд, на веб-странице показывается как один.
Тег <pre>позволяет обойти эту особенность и отображать текст как требуется разработчику.
Пример использования:

<pre>
                   //          //
                </pre>
            

                   //          //
