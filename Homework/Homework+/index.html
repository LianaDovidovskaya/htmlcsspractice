Примеры использования тегов

Теги dl,dt,dd

Элементы dl, dt, dd предназначенны для создания списка определений.
Каждый такой список начинается с контейнера dl, куда входит тег dt создающий термин и тег dd ,
задающий определение этого термина. Закрывающий тег /dd не обязателен, поскольку следующий тег сообщает о завершении предыдущего элемента.
Тем не менее, хорошим стилем является закрывать все теги.
Пример использования:

<code>
<dl>
<dt>Тег <tt></dt>
<dd>Контейнер <tt> отображает текст моноширинным текстом. Этот тег относится к группе тегов физического форматирования.</dd>
<dt>Тег <code></dt>
<dd>Тег <code> предназначен для отображения одной или нескольких строк текста, который представляет собой программный код.
Сюда относятся имена переменных, ключевые слова, тексты функции и т.д. Браузеры обычно отображают содержимое контейнера <code> как моноширинный текст уменьшенного размера.</dd>
</dl>
</code>

Тег <tt>
Контейнер <tt> отображает текст моноширинным текстом. Этот тег относится к группе тегов физического форматирования.
Тег <code>
Тег <code> предназначен для отображения одной или нескольких строк текста, который представляет собой программный код.
Сюда относятся имена переменных, ключевые слова, тексты функции и т.д. Браузеры обычно отображают содержимое контейнера <code> как моноширинный текст уменьшенного размера.

Тег pre

Элемент <pre> определяет блок предварительно форматированного текста. Такой текст отображается обычно моноширинным шрифтом и со всеми пробелами между словами.
По умолчанию, любое количество пробелов идущих в коде подряд, на веб-странице показывается как один.
Тег <pre>позволяет обойти эту особенность и отображать текст как требуется разработчику.
Пример использования:

            <pre>
            <kbd>   Contr    +     Alt        +     Shift   </kbd>
            </pre>
            
        

          Больше     пробелов     боооольше   !   !   !
